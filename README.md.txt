# 🎬 Cinema Tickets

Aplicación web para la gestión de entradas de cine.  
El proyecto está dividido en dos partes:

## 📌 Estructura del repositorio
- **api/** → Proyecto Backend (API REST) para la gestión de películas, usuarios y compra de entradas.
- **frontend/** → Proyecto Frontend en React para la interfaz de usuario.

## 🚀 Tecnologías
- **Backend:** C# .NET / Node.js (según implementación final)
- **Frontend:** React + Vite / CRA
- **Base de datos:** SQL Server / MongoDB (dependiendo de la configuración que uses)
